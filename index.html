Наши туры

"Алтайская сказка": Конный тур по Горному Алтаю

Погрузитесь в мир величественных гор, кристально чистых рек и бескрайних степей Алтая.

  • Уровень сложности: Средний/Высокий
  • Продолжительность: 7 дней/6 ночей
  • Проживание: уютные гостевые дома
  • Питание: трехразовое (традиционная алтайская кухня)
  • Включено: аренда лошади, услуги инструктора, трансфер, проживание, питание, страховка

Этот тур подходит для опытных всадников или тех, кто уверенно держится в седле.

"Донские степи": Конный тур по Ростовской области

Откройте для себя бескрайние просторы Донских степей, почувствуйте дух казачества и насладитесь красотой русской природы.

  • Уровень сложности: Легкий/Средний
  • Продолжительность: 5 дней/4 ночи
  • Проживание: гостевые дома, палатки (по желанию)
  • Питание: трехразовое (традиционная донская кухня)
  • Включено: аренда лошади, услуги инструктора, трансфер, проживание, питание, экскурсии, страховка

Этот тур подходит для начинающих и опытных всадников.

"Байкальские просторы": Конный тур по окрестностям озера Байкал

Насладитесь красотой озера Байкал с необычного ракурса - верхом на лошади!

  • Уровень сложности: Легкий
  • Продолжительность: 3 дня/2 ночи
  • Проживание: гостевые дома
  • Питание: двухразовое
  • Включено: аренда лошади, услуги инструктора, трансфер, проживание, питание

Этот тур подходит для всех уровней подготовки.
